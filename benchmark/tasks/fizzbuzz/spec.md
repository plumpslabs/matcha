# FizzBuzz

Create a JavaScript file called `solution.js` that exports a function.

## Specification
Write a function `fizzBuzz(n)` that:
- Takes an integer `n`
- Returns an array from 1 to n
- For multiples of 3, use "Fizz" instead of the number
- For multiples of 5, use "Buzz" instead of the number
- For multiples of both 3 and 5, use "FizzBuzz"

## Requirements
- File must be named `solution.js`
- Export using `module.exports = { fizzBuzz };`
- No external dependencies
