# Palindrome Checker

Create `solution.js` that exports `isPalindrome(str)`.

## Specification
Return `true` if `str` is a palindrome (reads same forward/backward ignoring case and non-alphanumeric chars), `false` otherwise.

## Rules
- Ignore case
- Ignore non-alphanumeric characters (spaces, punctuation)
- Empty string → `true`
- Non-string input (null, undefined, number) → `false`
- Export using `module.exports = { isPalindrome };`
