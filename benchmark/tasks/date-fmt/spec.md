# Date Formatter

Create a JavaScript file called `solution.js` that exports a function.

## Specification
Write a function `formatDate(date, fmt)` that:
- Takes a `Date` object and a format string
- Supports "YYYY-MM-DD" format (e.g., 2024-03-15)
- Supports "DD/MM/YYYY" format (e.g., 15/03/2024)
- Zero-pads single-digit months and days
- Returns the formatted string

## Requirements
- File must be named `solution.js`
- Export using `module.exports = { formatDate };`
- No external dependencies
