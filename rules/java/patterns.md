# Java Patterns

- **Spring Boot**: constructor injection, `@Service`/`@Repository` layering, `@ControllerAdvice` for errors
- **JPA**: `@EntityGraph` for N+1, `@Transactional` explicit boundaries
- Testing: JUnit 5 + Mockito + AssertJ
- Overlap check: before adding Maven/Gradle dependency, search `pom.xml`/`build.gradle`
