# Debounce Function

Create a JavaScript file called `solution.js` that exports a function.

## Specification
Write a debounce function `debounce(fn, delay)` that:
- Takes a function `fn` and a delay in milliseconds
- Returns a new function that delays invoking `fn` until `delay` ms after the *last* call
- If called multiple times, resets the timer each time

## Requirements
- File must be named `solution.js`
- Export using `module.exports = { debounce };`
- No external dependencies
