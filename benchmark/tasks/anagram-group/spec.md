# Anagram Group

Create `solution.js` that exports `anagramGroups(strs)`.

## Specification
Given an array of strings `strs`, group the anagrams together. An anagram is a word formed by rearranging the letters of another. Return array of arrays, each sub-array containing anagrams.

## Rules
- Order of groups and order within groups doesn't matter
- Each string belongs to exactly one group
- Non-array input → `[]`
- Export using `module.exports = { anagramGroups };`
